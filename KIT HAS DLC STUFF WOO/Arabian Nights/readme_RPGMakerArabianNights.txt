I. Introduction

Thank you for purchasing the Resource Pack, "RPG MAKER Arabian Nights Tileset Resource Pack". 
This Resource Pack contains a collection of middle-Eastern inspired tiles. This Add-on Pack is compatible with the full version of "RPG MAKER VX, RPG Maker XP, and RPG Maker VX Ace" and can be used with IG Maker to create your own game.

This explains the basic instructions of how to use the materials contained in this Resource Pack. Please read carefully before adding this music to your projects. 

------------------------------------------------------------

II. System Requirements

None
------------------------------------------------------------

III. About Resources in this Materials Add-on Pack

The resources in this Resource Pack are optimized for use with the full version of "RPG MAKER VX Ace" and the instructions below in "How to add the resources to your game" is written with RPG MAKER VX Ace in mind. If you are using a different product from the RPG Maker series, you'll need to manually import the resources by following the instructions in the help file of that product.

------------------------------------------------------------

IV. How to add the Resource Pack to your game

After the Resource Pack, "Arabian Nights Tileset Resource Pack" is completely downloaded to your computer, unzip it and place the folder in your desired location. 

Options in the Setup menu:

1. READ ME FIRST: 
You must click "READ ME FIRST"
- This read-me file will be displayed. 

2. Open RPG Maker VX Ace:
Open RPG Maker VX Ace and the project you wish to add the Resource Pack to. Select Tools -----> Resource Manager----->Graophics/Tilesets--------->Import-------->Choose the resource files and then import them. 

3. Set up the tilesets in the Database:
In the database, click the Tilesets tab. Click on the (...) next to each section and select the appropriate tiles. Note that the files names direct which section the tile should go in (for example, ArabianNights-TileB should go in the Tile B section).

4. High Five
The Resource Pack is now in your game. Wasn't that easy!
